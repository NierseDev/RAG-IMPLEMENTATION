TECHNICAL ARCHITECTURE
================================================================================

State Structure:
  debug.agentHierarchy = {
    mainAgent: {
      type: 'main',
      status: 'idle|running|completed',
      reasoning: string[],
      metrics: {
        duration, retrievedDocuments, iterations, confidence
      }
    },
    subAgents: [{
      id, type, status, reasoning[], result, metrics
    }],
    expandedAgents: { agentId: boolean }
  }

State Manager Actions (7):
  1. recordSubAgentSpawn - Create and log sub-agent
  2. recordSubAgentResult - Complete and store results
  3. updateMainAgentMetrics - Update main metrics
  4. updateMainAgentReasoning - Update reasoning trace
  5. updateAgentHierarchy - Replace full hierarchy
  6. toggleAgentExpanded - Toggle expand/collapse
  7. clearAgentHierarchy - Reset all agents

UI Components (3):
  1. AgentHierarchyPanel - Tree display with expand/collapse
  2. ReasoningTimeline - Chronological timeline with icons
  3. MetricsPanel - Real-time metrics aggregation

Colors:
  • Main: #667eea (Blue)
  • full_document: #8b5cf6 (Purple)
  • comparison: #06b6d4 (Cyan)
  • extraction: #10b981 (Green)
  • search: #f59e0b (Amber)

================================================================================
INTEGRATION POINTS
================================================================================

For Backend Integration:

When spawning a sub-agent:
  const agentId = await stateManager.dispatch('recordSubAgentSpawn', {
    agentType: 'full_document',
    context: 'Processing full document',
    reasoning: ['Initial analysis']
  });

When completing a sub-agent:
  await stateManager.dispatch('recordSubAgentResult', {
    subAgentId: agentId,
    result: 'Processing complete',
    reasoning: ['Analysis steps'],
    metrics: { retrievedDocuments: 10, confidence: 0.85 }
  });

When updating main agent:
  await stateManager.dispatch('updateMainAgentMetrics', {
    metrics: {
      duration: 5.5,
      retrievedDocuments: 15,
      iterations: 2,
      confidence: 0.92
    }
  });
